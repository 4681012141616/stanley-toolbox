CSS Carousel

A continuous carousel run on a single CSS animation

How it Works

No JavaScript

Just a linear infinite, 6-keyframe CSS animation with staggered starts for each section, block, or image

The relative timing depends on the number of carousel sections

Robustness

The carousel must run to the edges of the viewport, but works with any elements

Vertical scrolling works

Even nested carousels are possible using the same animation keyframes
